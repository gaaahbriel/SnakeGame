🐍 Snake Game

Um clássico Jogo da Cobrinha desenvolvido para fins de estudo e prática de lógica de programação.

🎮 Sobre o Projeto

Este projeto recria o famoso jogo da cobrinha, onde o objetivo é controlar a cobra, comer a comida e crescer o máximo possível sem colidir com as paredes ou com o próprio corpo.
